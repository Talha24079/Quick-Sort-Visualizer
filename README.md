# Quick Sort Visualizer

An interactive, step-by-step visualizer for the Quick Sort algorithm built entirely with HTML, CSS, and JavaScript. 

## Features
- **Zero Dependencies:** Pure vanilla JavaScript, CSS, and HTML. No Node.js, npm, or external libraries required.
- **Interactive Playback:** Pause, play, and step forward or backward through the sorting algorithm to understand exactly how Quick Sort works.
- **Adjustable Speed:** Use the speed slider to speed up the animation or slow it down to catch every detail.
- **Custom Input:** Enter your own custom array of numbers to see how Quick Sort handles them.
- **Premium UI:** A sleek, modern dark theme featuring glassmorphism and vibrant color gradients for the visualizing bars.

## How to Run

Because this project uses vanilla web technologies, running it is incredibly simple:

1. Clone or download this repository to your computer.
2. Navigate to the folder containing the project files.
3. Simply double-click the `index.html` file. It will automatically open in your default web browser (Chrome, Edge, Safari, Firefox, etc.).
4. You are ready to start visualizing!

## Usage
- Click **"Generate New Array"** to create a random array of 30 bars.
- Enter comma-separated positive numbers in the text box (e.g., `50, 20, 100`) and click **"Set Array"** to create a custom array.
- Click **"Sort"** to initiate the algorithm.
- Use the **Play/Pause**, **Prev Step**, and **Next Step** buttons to navigate through the algorithm manually.
- Adjust the **Speed** slider to make the visualization run faster or slower in real-time.
